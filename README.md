# Walmart Sales Analysis using SQL & Python

## Project Overview

This project analyzes 10,000+ Walmart sales transactions using SQL and Python to uncover sales trends, customer behavior, branch performance, and revenue insights. The analysis focuses on solving real-world business problems through data cleaning, exploratory data analysis (EDA), and advanced SQL queries.

---

## Objectives

- Analyze sales performance across branches and product categories.
- Identify customer purchasing patterns and payment preferences.
- Evaluate revenue and profit trends.
- Generate actionable business insights using SQL and Python.

## Data Processing

- Cleaned and transformed raw sales data using Python.

## Business Questions Solved

### SQL Analysis

- Most popular payment methods
- Highest-rated category per branch
- Busiest day for each branch
- Revenue by product category
- Sales distribution by shift (Morning, Afternoon, Evening)
- Most common payment method by branch
- Top-performing category by branch
- Revenue decline analysis (Year-over-Year)
- Customer rating analysis
- Branch-wise performance comparison
